# 📊 Sistema de Relatórios

Aplicação desenvolvida em Delphi para geração e impressão de relatórios do sistema principal.

## 📋 Sobre o Projeto

Este projeto tem como objetivo centralizar a geração de relatórios em um executável independente, facilitando a manutenção, distribuição e evolução dos relatórios sem impactar diretamente o sistema principal.

## ✨ Funcionalidades

- Emissão de relatórios gerenciais
- Impressão de documentos
- Visualização prévia dos relatórios
- Exportação para PDF
- Integração com banco de dados PostgreSQL
- Filtros personalizados para consultas

## 🛠️ Tecnologias Utilizadas

- Delphi
- FireDAC
- PostgreSQL
- FastReport
- Git
